# Debugging e Tratamento de Exceções em Java
# Sobre o projeto
Um projeto com foco em cadastro de usuários com, um CRUD simples para melhor aproveitamento, com tratamento
de algumas exceções focando no estudo e utilizando a ferramenta de debug para aprofundar em como cada coisa]
funciona

# Funcionalidades
* Criação de usuários com tratativa de exceções para evitar nomes e e-mails inválidos.
* Atualização do usuário
* Exclusão de usuários
* Busca de usuário por meio do ID
* Listagem de usuários

# Tecnologias Utilizadas
* Java
* Git e Github

# Extras
Criei um repositório git e um hospedei no github para aprender mais e manter em pratica o git e como funciona os commits,
push e pull trabalhando de diferentes máquinas. Aprendi muita coisa com esse projeto e o foco continua.
